Welcome

Overview
This package contains several products which allow a variety of effects and controls to be added to web pages. The effects and controls allow more information to be placed within an area of a web and at the same time provide visitors with a dynamic and interactive experience.
Implementation and Set Up
For each product there is a GettingStarted.htm document which contains detailed instructions on how to add each control/effect to a web page. In addition a full Parameter Reference is provided which will enable each product to be configured / customised providing the 'look and feel' required.
