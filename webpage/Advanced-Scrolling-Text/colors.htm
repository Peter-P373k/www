Color Definitions

For any color parameter the color may be specified in any of the following formats:-

  • Standard HTML Hexadecimal Color Notation
  • Color Name
  • Decimal RGB (Red,Green,Blue) Notation

Standard HTML Hexadecimal Color Notation

Color values may be supplied in the same format as colors are defined within an HTML page.
ie. in the range and format from "#000000" to "#FFFFFF".

This is a Red,Green,Blue color model where each component is defined by a 2 byte (character) Hexadecimal number.

Decimal RGB (Red,Green,Blue) Notation

Color values may be supplied in Red, Green, Blue Format where each component is specified by a decimal integer in the range 0 to 255. Each value should be separated by a , (comma) character.
