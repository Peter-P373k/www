Getting Started

Overview
The Image Viewer provides a very efficient way to display multiple images within a small area of a web page. The style and size of the view area are set by setting the values of a series of parameters. Any number of images may be added to the viewer. With Configurator tool setting the parameters is easy. Simply select and the settings you require and the Configurator will provide you with the code to copy and paste into your page.
The Easy Way to Add the Image Viewer to your page.
The quickest and easiest way to implement the image viewer function within your page is to use the online configurator tool. Here you select the size and settings you would like, add your images and click the submit button. The configurator will then provide you with the HTML code that you then add to your page (ie. Copy and Paste).

Use the Online Configurator Now »

Once the HTML coded is added to your page upload the two files "jpimageviewer.js" and "jpimageviewerhere.js" to your web server placing them in the same directory as your web page. These files can be found in the same directory as this GettingStarted document. These two files contain the CSS and JavaScript code necessary to operate the Image Viewer function. They should not be modified in any way.
Advanced Users
If you have a good understanding of HTML and wish to set the parameters by hand then rather than use the Configurator Tool you can also construct the parameter list yourself. The documentation provides a full parameter reference and describes the effects in more detail.
