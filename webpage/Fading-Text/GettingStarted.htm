Getting Started

Overview
The Advanced Fading Text software enables you to display all your adverts in just one area of your page. No longer will those adverts swamp your real content.

Set Parameters
The parameters (PARAM tags) dictate the content, colors, fonts and operation of the Advanced Fading Text. The VALUE of the parameters (added in the previous step) should be adjusted to provide the desired result.

For the meaning and effect of each parameter see the Parameter Reference here


Add the Messages to be Displayed
The Advanced Fading Text will load the messages from a data file specified by the "dataURL" parameter. Above this parameter is set to a value of "data.txt". The dataURL does not have to be a static text file but can be (if required) a server side script which generates the content dynamically
(eg. <param name="dataURL" value="data.php"> where data.php would be a PHP script which provides the content upon request).

Simply add the messages to a file specified by the dataURL parameter. In addition to text it is also possible to add Tags which vary properties like:-
  • Colors
  • Fonts
  • Links
  • Pause Time
  • Alignment
For more information on these see Data File Special Tags here

Once complete, place the data file in the same directory as the web page

The ./examples/ directory of this package contains several data files.

Copy Files
From the ./AppletFiles/ directory of this package copy the following files and place them in the same directory as your web page:-
  • AdvFadeText.jar
  • jpapplet.js
